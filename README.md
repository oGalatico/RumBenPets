# RumBenPets

Pets plugin for your MCPE Server. Works with ImagicalMine, PocketMine-MP and Genisys.

Forked and extended for RumBenCraft

- All kinds of animal (Ocelot, Wolf, .... ) (need supportive for Entity)

- Pets will follow you!

- Name your Pets :)
